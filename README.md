# Montha Help Desk Backend

A Flask-based backend application developed to handle emergency reports during natural disasters like cyclones and floods.

## Tech Stack
- Python
- Flask
- MySQL

## Features
- Submit emergency reports (POST API)
- Fetch all reports (GET API)
- MySQL database integration
- Secure database user configuration
- Simple HTML form for testing

## API Endpoints
- GET /reports – Fetch all emergency reports
- POST /report – Submit a new emergency report

## How to Run
1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python app.py
